# 03. Goal time

## 問題

マラソン大会において、ある選手のスタート時刻とゴール時刻が与えられます。
記録を計算し、出力してください。

## 入力

テキストファイルに以下の2行のデータが渡されます。
1行目がスタート時刻、2行目がゴール時刻です。
hhは時、MMは分、ssは秒で、それぞれ2桁の数値です。一桁の場合は先頭に"0"が付与されます。

```text
hh:MM:ss
hh:MM:ss
```

## 出力

hh:MM:ssの形式で出力してください。

## 例：

入力：
```text
9:00:00
11:03:22
```

出力：
```text
02:03:22
```

## 制約

スタート時刻とゴール時刻は日付を跨ぎません。
